# UI Blocks & Components

Reusable UI components.

## Block Definition

```xml
<Block className="my-component">
  <h2 module="html">{Title}</h2>
  <p module="html">{Description}</p>
  <Action name="myAction">
    <Content>Click Me</Content>
  </Action>
</Block>
```
